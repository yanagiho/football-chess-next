## 1. 共通仕様（スマホ・PC共通）

### 1-1. ターンフェーズと操作フロー 【v3.0更新】

```
[TURN_START] → 「Turn X」演出(1秒) → 入力不可
     ↓
[INPUT] → タイマー60秒スタート → 操作受付
     │
     ├── コマタッチ → ドリブル先タップ → 命令確定
     ├── ボールタッチ → パス先/スルーパス先/シュート先タップ → 命令確定
     ├── パス成功 → ボールが味方に移動 → 受け手のボールをタップで次のパス
     │
     └── [確定ボタン] or [タイマー0秒]
          ↓
[WAITING] → COM思考中/相手待ち → 入力不可
          ↓
[EXECUTION] → スナップショット巻き戻し → リプレイ再生
     │   全コマ同時移動(0.8秒) → タックル/ファウル演出
     │   → FlyingBallでパス/シュート順番に飛行 → パスカット/オフサイド
     ↓
[EVENT] → ゴール/ファウル等の特殊演出 → 入力不可
          ↓
[TURN_END] → 0.5秒 → 次ターンの TURN_START へ
```

### 1-2. ボール操作（コマ/ボール分離タッチ） 【v3.0更新】

**ボール保持者をタッチ（コマ本体でもボールアイコンでも同じ） 【v3.1更新】:**
→ コマの近くにアクション選択メニューを表示:
  [⚽ パス] [🏃 ドリブル]
メニューはコマの上側-70pxにHexBoard transform内に配置（ズーム追従）。
z-index: 200。onPointerDownで即座に反応。

**「パス」選択後のタップ先自動判定:**
- 味方コマ → パス（PASS_BALL: ボール即座に移動、パス元命令済み）
- 空きHEX → スルーパス（THROUGH_PASS: ボールがHEXに飛ぶ）
- シュートゾーン → シュート
ガイド:「味方=パス / 空きHEX=スルーパス / ゴール方向=シュート」

**チェーンパス:** パス成功後、受け手のボールをタップして再びパスモードに入るだけ。
「キープ」ボタンやポップアップは不要。パスを出したコマは命令済み（移動不可）。

**ボール非保持者のコマをタッチ:**
→ 移動モード(4HEX)。ガイド:「移動先をタップ」。矢印: 白実線。

**ボールアイコンの表示:**
- コマ右上に配置、コマの60%サイズ（約40px）
- 白い丸 + 五角形パターン + drop-shadow(0 0 6px rgba(255,255,255,0.8))
- 常時パルス: fcms-ball-pulse-gentle (2秒周期, scale 1→1.08)
- Piece.tsx 内で `onBallClick` コールバック付きの別要素としてレンダリング

## 5. 解決アニメーション（EXECUTION再生） 【v3.0更新】

### 5-1. リプレイ再生方式 【v3.0更新】

ターン確定後、盤面を `turnStartSnapshot` に巻き戻してからリプレイ再生する。

| タイミング | 内容 |
|---|---|
| 0ms | スナップショット巻き戻し(SET_DISPLAY_PIECES) |
| 300ms | APPLY_ENGINE_RESULT → CSS transition で全コマ同時移動(0.8秒) |
| 1100ms | Phase1: 競合⚔/タックル💪エフェクト(Canvas) + CenterOverlay |
| 1600ms | Phase2: ファウル🟨エフェクト + CenterOverlay |
| 2100ms | Phase3: パス/シュートをFlyingBallで1つずつ順番に再生 |
| ～ | Phase4: パスカット/オフサイドエフェクト + CenterOverlay |
| ～ | 軌跡0.8秒表示 → クリア → ゴール演出 or TURN_END |

全体のアニメーション時間はSettingsContextの速度設定(0.5x/1x/2x)で除算される。
安全弁: 8秒タイムアウト(SAFETY_TIMEOUT_MS)。

### 5-2. FlyingBall（ボール飛行アニメーション） 【v3.0追加】

| 項目 | 値 |
|---|---|
| サイズ | 24px |
| 見た目 | 白い丸 + 五角形パターン(SVG) |
| z-index | 200（コマより上） |
| 回転 | SVG animation fcms-ball-spin 0.5s infinite |
| 光彩 | パス=青, シュート=赤, スルーパス=シアン (drop-shadow) |
| 飛行時間 | 200-500ms（距離に応じて自動調整、速度設定で除算） |
| 実装方式 | useRef + 直接DOM操作 + CSS transition |

再生順序:
1. 軌跡線をOverlay Canvasに描画（ボール飛行と同時に線が見える）
2. FlyingBall: 初期位置にジャンプ → reflow → 目的地へCSS transition
3. 到着 → onComplete → Promise resolve → 次のイベントへ
4. パスチェーンは1本ずつ順に飛行（A→B, B→C と連続）
